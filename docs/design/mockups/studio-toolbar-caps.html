Wave 5 · CAPS toolbar redesign

Studio · CAPS — Capture · Adjust · Preview · Publish · Share

Mnemonic. Today's bottom toolbar reads Camera · Refine · Preview · Publish · Share. Renaming Camera → Capture and Refine → Adjust lands a clean acronym — CAPS — that practitioners can repeat back to themselves while learning the workflow.

Visual upgrade. Pill geometry goes fully rounded (was 12px square corners). Each glyph drops into its own dark sub-cell, with a 10pt uppercase label below. Slim coral arrows replace the chunky triangles. Share switches to the three-dot network glyph. Active vs dim state mirrors today — the label dims with its glyph.

  • Pill: BorderRadius.circular(999), color: surfaceRaised, BoxShadow(0,2,8,rgba(0,0,0,0.25)).
