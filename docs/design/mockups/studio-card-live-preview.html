Spec

  • Preview: 200 × card-height (currently 132pt min). Edge-to-edge with the card's left + top + bottom borders. Object-fit cover, centred.
  • Source: same line-drawing H.264 the web player serves (line_drawing_url). Muted, autoplay, loop. Already on disk via the converter — no extra fetch.
  • Back nav: 32 × 32 circle, top-left at (8, 8). Coral icon stroke (--brand-default) on a translucent dark backdrop with a subtle coral border. Tappable.
  • "Playing" pill bottom-right of preview — pulsing red dot + tiny label so the practitioner knows it's a live tile vs a still.
  • Right column shrinks to card-width − 200 with normal padding. Title + customised-dot + gear sit on row 1; dose summary row 2; notes summary row 3.
  • Rest variant: keep the sage glyph (no video to play); same rectangle, no live pill, no back nav (rest periods don't have a preview to nav into).
  • Performance: pause off-screen previews; pause when the editor sheet is open over the Studio screen. Limit concurrent decoders if > 4 cards visible.
