Region legend

Every outline in the gallery above maps to a real DOM id / class in the web-player codebase. Region names are quoted verbatim from web-player/index.html + styles.css + app.js so “the matrix-eta” means the same thing to design and engineering. Grouped by outline family.

progress + time (coral)

plan-bar [all states] Top title strip. Holds plan-title (session title) and header-client (client name subtitle).
progress-matrix [all states] Container row of pills with left-edge fade + auto-centering on the active pill. Canonical source: docs/design/mockups/progress-pills.html.
pill [all states] Each exercise / rest slot. Empty idle, coral when active (pulse-glow) or completed, sage when rest. Holds pill-fill (live timer fill bar).
matrix-col.is-circuit [circuit] Vertical stack of pills — one row per cycle — sitting inside a coral tint band. Adjacent circuit columns seam together.
matrix-eta [all states] 3-number time row — matrix-eta-current (active-slide remaining, coral) · matrix-eta-total (whole-plan remaining, white) · matrix-eta-finish (wall-clock finish, muted).
progress-matrix-chevron [scrubbing] Snap-back chevron that appears when the matrix is manually scrolled off-centre. Auto-hides after 4s idle. (Not drawn in any of the 5 frames above — worth knowing.)

card + media (blue)

card-viewport [all states] Swipeable media container. Hosts card-track + injected swipe-hint. Pan-Y only — vertical scroll passes through.
exercise-card [all states] One slide in the deck — data-index carries its position. 100% of viewport width; the track scrolls by negative-translating.
card-inner [all states] Rounded surface panel inside the card. Hairline border, no shadow (D-07 flat). Holds card-media + card-body.
card-media [all states] Media pane — the video or img element lives here. Tap dispatches by mode: prep → skip, running → pause, paused → resume.
mute-toggle [running] Speaker-icon button, top-right of card-media, only rendered when the exercise ships with audio. Flips runtime isMuted.

metadata (violet)

card-body [all states] Bottom panel of the card — holds position kicker, exercise name, treatment-seg, and the luxurious detail line.
card-position [exercise] 11pt coral uppercase kicker (“Exercise 5”). Omitted on rest cards.
card-exercise-name [exercise] Exercise title. Montserrat 18pt 700, letter-spacing -0.02em.
treatment-seg [exercise] Line · B&W · Original segmented control. Tapping switches which url is played. Disabled segments show a lock glyph (consent absent).
active-slide-detail [all states] Luxurious grammar line — 3 sets · 10 reps · 5s hold (standalone), 10 reps · 5s hold (circuit; sets omitted, rounds carried by matrix), 30s rest (rest).
card-notes [exercise] Optional free-text note from the practitioner. Not drawn in the 5 frames above — appears only when slide.notes is set.

navigation (green)

nav-bar [all states] Bottom row hosting prev / dots / next.
btn-prev / btn-next [all states] Manual chevron nav. btn-prev disabled on slide 0; btn-next advances / completes.
nav-dots [all states] Slide indicator dots. Active dot is coral elongated. (Hide automatically past 10 slides per the current spec.)
peek-panel [teaching, scrubbing] Centered dark-card showing peek-name + peek-meta. Auto-shows for 2s on preview start (teaching peek R-09); reappears during long-press matrix scrub.
swipe-hint [first load] “Swipe to navigate” hint injected into card-viewport on first card only.

state-specific overlays (amber)

prep-overlay [prep] Big coral countdown (prep-overlay-number) fading the last 200ms of each second. Synchronised with matrix-eta-current + active pill is-prep-flashing.
media-pause-overlay [paused] Centered dark-disc play triangle overlay. Injected into every card — toggled visible only on the active paused slide.
rest-card [rest] Variant of card-inner with a sage gradient, sage rest-icon, rest-title, and rest-next-up (“Next up: X”).
rest-media [rest] Rest variant of card-media — gradient surface hosting rest-display (icon + title + next-up subtitle).
circuit-indicator [legacy / not rendered] “Circuit · Round 2 of 3 · Exercise 1 of 3” bar. Marked display: none !important in styles.css (item 14 of mobile parity) — circuit context now expressed spatially by matrix-col.is-circuit. Drawn on state 5 for reference if we decide to bring it back.
workout-complete [final state] Full-screen overlay shown when the last slide completes. Holds circle-check icon + total time + Close button. Not drawn in the 5 frames above.
start-workout-btn [pre-start] Floating coral pill at bottom-center that kicks off the 15s prep countdown on slide 0. Not drawn above (all 5 states assume the workout is running).
