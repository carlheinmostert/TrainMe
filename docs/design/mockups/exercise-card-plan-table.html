13Circuit context · cycles & rows are independent

Cycle count is a property of the circuit, set count is a property of the exercise — they edit separately. Two reconciliation rules: cycles > rows → the last set repeats for the remaining rounds (top card). cycles < rows → the first N sets play and trailing rows are marked "won't play" (bottom card).

15Web-player · rep-block stack

Vertical stack from a 3-set pyramid (10 @ 12.5 kg / 8 @ 15 kg / 6 @ 17.5 kg). Each set group carries its weight chip. Rest is a clean sage divider — no text inside the pill (the duration belongs in the active timer chip, not the static stack). The upcoming set's weight rides as a separate pill to the right of the rest divider. The final rest divider carries an "End of exercise" marker (option c) so the post-set breather is honoured visually but the absence of a next set is explicit.
