Behaviour notes

Library import inside Camera. Tapping the bottom-left library icon opens the iOS multi-select photo picker. When the user picks N photos, each photo flows through the SAME pipeline as a captured photo — individual peek-box animations fire one per import, conversion queue handles them sequentially, no batched silent ingest.
Slide-up-to-lock gesture. The lock target appears the moment recording starts (∼200ms fade-in). While the finger is pressed on the shutter, vertical drag-up onto the lock target snaps the lock — haptic confirmation, shutter inner morphs from red-dot to white-square, recording continues with finger released. To stop: tap the morphed shutter.
Permanent hint copy. Idle: Tap for photo · Hold for video · Slide ↑ to lock at 55% opacity. Pressed-recording: copy swaps to Slide finger ↑ onto 🔒 to lock. Locked: copy swaps to Tap ⬛ to stop.
Right-edge vertical lens pills. The four pills (.5 / 1× / 2 / 3) stack vertically right of the viewfinder. 44×44pt pills, 8pt gap, vertically centred. Active pill inverts (white background, black text). Doesn't overlap the lock-drag corridor.
Bottom-left library icon. 44×44pt rounded button, ~16pt from screen edges. Sits well below the peek box (which stays mid-left). Icon is the standard photo-stack glyph from iOS Photos.
