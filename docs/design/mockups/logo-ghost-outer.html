Signed off 2026-04-20. Matrix-only + lockup share the same 11-element geometry. Source of truth for HomefitLogo.tsx, homefit_logo.dart, and buildHomefitLogoSvg().

Matrix only · viewBox="0 0 48 9.5"

Default variant. Use in header brand-marks (paired with a text wordmark), footers, favicons, app icons, tight chrome. Aspect ratio 48:9.5 (~5.05:1).

Lockup · viewBox="0 0 48 14"

Hero variant. Wordmark row sits above the matrix, stretched via textLength="48" so it aligns with the matrix width at any render size. Matrix is identical geometry, translated +4.5 on Y. Use on sign-in screens, OG cards, email templates, printed share cards.
