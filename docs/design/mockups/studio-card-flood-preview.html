Spec

  • Card: full width × 152pt tall, border-radius: 16pt, no border. Focused state = box-shadow: 0 0 0 2px coral (outer ring, doesn't push the video).
  • Video: position: absolute; inset: 0; (= no margin from card outline), object-fit: cover. Same line_drawing source pulled from the converter.
  • Scrim: bottom-up gradient covering the lower 70% of the card. rgba(0,0,0,0.85) at the floor → 0.55 at 40% → fully transparent at the top. Guarantees title legibility over any frame.
  • Title text + summary text use a soft text-shadow: 0 1px 2px rgba(0,0,0,0.6) as a second-line legibility layer for bright frames.
  • Gear: top-right, 32 × 32 circle, rgba(0,0,0,0.45) backdrop with blur(6px). Customised dot = 8pt coral with a 1.5pt black border so it doesn't bleed into bright frames.
  • Whole card tap → opens editor on Plan (preserves the recent change). Long-press → replace media. Inner taps for gear / plan-row / notes-row keep their own hit zones via Flutter's gesture arena.
  • Rest variant: sage linear-gradient swap-in, big translucent glyph centred. Lighter scrim (no need for legibility layer over a quiet gradient). No video controller.
  • Performance: same single-controller-per-card model. Pause when off-screen (via VisibilityDetector — punt to v2 if too heavy).
