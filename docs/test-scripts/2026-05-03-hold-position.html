2026-05-03 · Hold position 3-mode (Wave 43)

Device QA — Hold position 3-mode

Purpose: verify the new per-set hold position control. Each set gets a small 3-pill segmented control beneath the existing Hold stepper. The chosen mode controls how hold_seconds contributes to the per-set duration on both surfaces (mobile preview + web player at session.homefit.studio).

The three modes:

Wire valueUI labelMath contribution
per_repPer repreps × hold (legacy contract)
end_of_setEnd of set ✓1 × hold (new default)
end_of_exerciseEndhold on the LAST set only; 0 elsewhere

Migration: existing rows with hold_seconds > 0 were backfilled to per_rep so already-published plan durations stay byte-stable. Rows with hold_seconds = 0 use the new default (math identical in all 3 modes when hold is 0). New sets default to end_of_set.

Surfaces: mobile editor sheet (Plan tab) is the only place to change the mode — web is consumption only. Mobile preview + web player both honour the chosen mode in their per-set duration math.

Out of scope: portal doesn't render hold-timing data; types updated but no UI surface.

1 · Editor UI

  • 1. Open any video exercise in the editor sheet → Plan tab. Tap the Hold cell on any set to expand its inline editor. Confirm the existing chip row (Off · 5s · 10s · 30s · 60s) is still there, AND a NEW horizontal row of 3 coral-bordered pills sits directly beneath it: [ Per rep ] [ End of set ] [ End ]. On a freshly-added set the highlighted pill is End of set (with a small white check inside).
  • 2. With hold = 5s and reps = 12, tap Per rep. The pill highlights coral. Note the ETA at the top of the Preview tab — should grow noticeably (the per-set physical adds 12 × 5 = 60s instead of just 5s). Close the editor sheet, re-open the same exercise, return to Plan tab → tap the same Hold cell. The selection should still read Per rep (persistence check).
  • 3. Tap End. ETA shrinks back. With a single-set exercise the math is identical to End-of-set; with a multi-set exercise (3+ sets), only the LAST set's hold contributes — non-last sets contribute 0. Confirm by adding a second + third set with the same Hold mode set to End: the ETA grows by reps × per_rep + breather per non-last set, plus hold only once at the end.
  • 4. Tap Add Set at the bottom of the Plan table. A fresh set appears. Tap its Hold cell → expand the editor → confirm the End of set pill is selected by default (this is the "new sets default" check, regardless of what mode the previous set is on).
  • 5. Set hold = 0 (tap the Off chip). The 3 pills should fade to ~50% opacity (dimmed). Tap the pills anyway — the selection visibly updates (the highlighted pill changes), but it's clear the choice has no current duration impact. Bump hold back to 5s → pills regain full brightness, the previously-selected mode is preserved.
