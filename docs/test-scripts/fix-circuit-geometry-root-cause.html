2026-05-15 - Circuit geometry (root-cause fix)

Device QA - Circuit lane geometry + tracer animation

Background. The lobby's coral spiral tracer that runs around a circuit's frame has been broken in the iOS embedded preview for five prior attempts. Today's QA also surfaced a second symptom: in the PDF preview modal, the circuit's coral border closes too early - Exercise 3 of a 2-exercise CIRCUIT A falls outside the visual frame.

Confirmed root cause: frame.offsetHeight was read ONCE inside renderCircuitLanesFor() on entry - before the lazy-loaded hero <img> elements had decoded. The frame at that moment was sized to placeholder skeleton heights. The static SVG lane rect AND the tracer's d attribute were both derived from this stale geometry, so the 30-frame retry loop in PR #322 was retrying getTotalLength() on the SAME stale path. One root cause, two symptoms.

Why each prior fix failed:
- PR #257/#258 - restructured DOM frame markup, but never addressed the measurement timing.
- PR #259 - introduced renderCircuitLanesFor with the single-measurement bug.
- PR #260 - switched CSS animation -> WAAPI (correct hardening, doesn't fix geometry).
- PR #317 - added 1 rAF retry of getTotalLength() (retrying the wrong thing).
- PR #322 - added 30-frame poll + CSS-vs-WAAPI hardening (still retrying the wrong thing).

The durable fix:
- await document.fonts.ready for font metrics.
- await Promise.all(...img load/error) for every hero image in the frame.
- Re-read offsetWidth / offsetHeight only after both signals resolve.
- ResizeObserver attached BEFORE measurement, per-circuit, so any subsequent layout shift (orientation flip, treatment toggle, late image decode) triggers a clean recompute.
- Existing 30-frame poll + WAAPI hardening kept as defence-in-depth.

R-10 parity: both web-player/lobby.js and app/assets/web-player/lobby.js edited byte-identical (sw.js cache name bumped via build.sh on Vercel deploy).

In scope: circuit lane rect + tracer animation in embedded preview + live web staging + PDF preview modal. Cold-cache path. Orientation flip.

Out of scope: any non-circuit lobby behaviour, prep hero treatment, photo body-focus, filmstrip sharpness - all separate.
