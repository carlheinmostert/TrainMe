Swipe duplicate · Studio exercise duplication

Swipe-right-to-duplicate exercise (deep copy)

Right-swipe on a Studio exercise card immediately duplicates it below the original. Deep copy: fresh UUID, all metadata + all media files physically copied. Undo via R-01 SnackBar. Rest periods are excluded. 5 items.

Studio · swipe-right-to-duplicate

  • 1. Right-swipe on a video exercise duplicates it below with same thumbnail + metadata.
    1. Open an existing session with at least one video exercise in Studio.
    2. Swipe RIGHT on the video exercise card. A coral "Duplicate" label with copy icon should reveal on the left as you swipe.
    3. Complete the swipe. The card snaps back into place and a new card appears directly below it.
    4. Verify the duplicate has the same name, thumbnail, reps, sets, hold, and notes as the original.
    5. Verify the "Duplicated · Undo" SnackBar appears at the bottom.
    6. Verify position numbers updated: if original was #3, duplicate is now #4 and subsequent exercises shifted.
  • 2. Right-swipe on a photo exercise duplicates it with same thumbnail + metadata.
    1. In a session with a photo exercise, swipe RIGHT on the photo card.
    2. Duplicate appears below. Verify thumbnail matches the original photo.
    3. Expand both cards and compare all fields (name, reps, sets, hold, notes). They should match.
  • 3. Undo removes the duplicate cleanly.
    1. Duplicate any exercise via right-swipe.
    2. Tap "Undo" on the SnackBar within 5 seconds.
    3. The duplicate disappears. Position numbers revert to their original values.
    4. Leave Studio, come back. Verify the duplicate is truly gone (not just hidden).
  • 4. Duplicate inside a circuit stays in the same circuit.
    1. Create or open a session with a circuit (two or more linked exercises).
    2. Right-swipe on one of the circuit exercises.
    3. The duplicate appears below the original, still inside the same circuit (circuit rail bracket includes it).
    4. Verify the circuit header still shows the correct exercise count.
  • 5. Rest period: swipe does nothing.
    1. In a session with a rest period, try to right-swipe on the rest bar.
    2. The rest bar should not duplicate. (Rest rows only support swipe-left-to-delete.)
