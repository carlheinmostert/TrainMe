2026-05-14 · Lobby + deck gear unification

Lobby settings unify — QA

Scope: the gear popover on the lobby and the gear popover on the deck now render the same unified panel — treatment selector, body-focus toggle, "Reset to practitioner" button. All three controls write a plan-scoped override (no more per-exercise client overrides), so toggling on either surface sets the scope for the whole session.

Principle: practitioner-set per-exercise preferred_treatment + body_focus stay sacred; the client's plan-scoped override layers ON TOP of every exercise. Reset returns each exercise to its practitioner-set default — so a mixed plan (Ex 1 = B&W, Ex 2 = Line, Ex 3 = Colour) snaps back to mixed on reset.

What's NOT in scope: the SHARE/PDF export pipeline (separate PR), photo three-treatment thumbnail variants (parallel work), Studio cards / filmstrip / Studio card hero (practitioner-facing, untouched by client override). Test 7 verifies those stay unaffected.

Prerequisites

  • PR feat/lobby-settings-unify merged to staging.
  • Visit https://staging.session.homefit.studio/p/<plan-id> for a plan with at least 3 video exercises.
  • For best coverage, use a plan where the practitioner has set MIXED preferred_treatment per exercise (e.g. Ex 1 = B&W, Ex 2 = Line, Ex 3 = Colour). If you don't have one, set one up in the trainer app or via the mobile preview.
  • Plan with both grayscale and original consent granted (so no consent locks block the test flow).

Lobby gear popover

  • 1. Lobby gear popover content matches the deck gear popover content. Open the lobby. Tap the gear icon in the bottom-right CTA bar. The popover opens upward. Expected: a single For this session panel with three controls — Treatment (Line / B&W / Colour), Body focus toggle, and Reset to practitioner button. Identical visual layout to the deck's gear popover (test 5 confirms the deck side).
  • 2. Plan-scoped treatment: pick Line on lobby gear → every row renders Line. With a mixed-treatment plan, tap the gear → tap Line. Popover closes. Expected: every hero row in the lobby (including ones the practitioner set to B&W or Colour) renders the line drawing. Same on the deck after tapping Start Workout.
  • 3. Plan-scoped treatment: pick Colour on lobby gear → every row renders Colour. With the same plan from #2, open gear → tap Colour. Expected: every hero renders the colour raw footage. Verify the deck does the same.
  • 4. Reset to practitioner → lobby returns to the mixed-treatment original state. After #2 or #3 has set a global treatment, open the gear and tap Reset to practitioner. Expected: Ex 1 returns to B&W, Ex 2 to Line, Ex 3 to Colour (whatever the practitioner set per-exercise). The Reset button greys out (no overrides remain).

Deck gear popover (post Start Workout)

  • 5. Deck gear popover visually identical to lobby gear popover. Tap Start Workout. On the first slide, tap the gear icon in the top-right chrome stack. Expected: the popover heading reads For this session; below: Mute toggle, Treatment (Line / B&W / Colour), Body focus toggle, Reset to practitioner button. (The lobby popover omits the Mute row because there's no audio playing pre-workout; otherwise identical content.)
  • 6. Per-exercise gear: tapping a treatment applies session-wide (NOT just current exercise). Reset state from #4 (or fresh reload). Mid-workout, tap the gear → tap B&W. Swipe to the next exercise. Expected: both the previous AND the next exercise render in B&W. Tap gear again → tap Line. Swipe back. Expected: both render in Line. (Pre-PR: tapping a treatment only affected the current exercise.)
  • 7. Per-exercise gear → Reset to practitioner → all exercises revert to their practitioner default. After #6 has set a global treatment, tap gear → tap Reset to practitioner. Swipe through all exercises. Expected: each exercise returns to its practitioner-set per-exercise treatment (mixed if the plan was mixed).
